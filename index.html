Types of Software

There are two different types of software,

  • system/utility
  • application

Both are important for a functioning computer that will provide utility to a user. I will give a high level overview of both types, along with specific examples of each.

System and utility software typically interact directly with the hardware. This type of software consists of:

  • operating system
  • firmware
  • BIOS etc.

Application software runs on top of the system software, and communicates with system software to interact with the hardware. Types of application software includes:

  • spreadsheets
  • browsers
  • websites, etc

Read the software wikipedia to learn even more about the different types of software.
